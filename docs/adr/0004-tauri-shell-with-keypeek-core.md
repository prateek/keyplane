# Tauri shell with KeyPeek core logic

The app will reuse and reshape KeyPeek's Rust protocol and domain code while replacing KeyPeek's egui interface with a Tauri shell and web frontend. This keeps KeyPeek's VIA, Vial, ZMK, HID, key-label, device-discovery, and matrix logic as the starting point while making overlay and settings UI work in web technology.
